Safe Crossing

This project is about determining which crosswalk a pedestrian is about to cross with a mobile phone and optionally hardware devices installed on the intersection.

View project on GitHub

The project

Our goal is to help visually impaired people safely cross roads, by providing them with more precise routing and orientation information. This is especially important for crosswalks that are not equipped with ratchet tickers. We propose a solution that consists of three major parts:

  1. Routing and coarse positioning.

  2. Fine positioning on high-risk areas, using special equipment and an app to provide the user with the necessary information.

  3. App providing the user with real-time feedback.

This combination of technologies would allow us to precisely track users’ positions and ensure their safety during road crossing through real-time accessible feedback. Using the most promising technologies explored by our team, Imec is planning to develop an accessible road crossing system that will improve peoples’ lives.
